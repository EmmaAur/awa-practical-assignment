Installation (this same text can be found ): 

You need MongoDB and Compass to use and keep a track of the database
Create a connection to "mongodb://localhost:27017" and connect the database
The projects database is called "projectwork"

Download the master branch from GitHub and install the next packages:
1. Go to the root folder of the project and run "npm install"
2. Go to the client folder and run "npm install"
3. Go to the server folder and run "npm install"
4. after that the project can be run from the root folder with the command "npm run dev"
